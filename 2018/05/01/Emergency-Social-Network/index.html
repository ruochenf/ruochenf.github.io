Emergency Social Network

Ruochen Fang
2018-05-01

Emergency Social Network is an application to support small community of civilians affected by emergency situations caused by natural disasters.

Technical Stack

  • Front-end
    • Angular
  • Back-end
    • Node.js
    • Express framework
    • Socket.io
  • Database
    • MongoDB
  • Testing && Code Quality
    • Mocha
    • Circle CI
    • Code Climate
    • Swagger.io

Main Features

  • Public, private and group chats

  • Announcements page monitored by Administrators

  • Search functionality by messages, users, status

  • Identify Emergency and calamities events nearby

  • Volunteering options
